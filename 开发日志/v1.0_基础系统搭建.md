# v1.0 基础系统搭建

**日期：** 2026年4月

## 概述

完成智能文档问答系统（Smart QA）的核心架构搭建，实现从文档上传到问答的完整流程。

## 技术栈

- 后端：Python 3.13 + FastAPI
- 向量数据库：ChromaDB（持久化存储）
- Embedding：paraphrase-multilingual-MiniLM-L12-v2
- BM25：rank-bm25 + jieba 中文分词
- LLM：DeepSeek API（deepseek-v4-flash）
- 前端：Streamlit

## 实现功能

### 文档处理
- 支持 PDF、DOCX、TXT、Markdown 四种格式上传
- 自动文本切片（500字符，100字符重叠）
- 句子边界感知切片，避免截断句子
- TXT 文件多编码回退（utf-8 → gbk → gb2312 → latin-1）

### 检索系统
- 向量检索：ChromaDB + SentenceTransformer 余弦相似度
- BM25 关键词检索：jieba 分词 + rank-bm25
- 混合检索：Reciprocal Rank Fusion (RRF) 融合两种检索结果
- HyDE 查询改写：先生成假想答案，用答案做检索（提升语义匹配）

### 问答生成
- DeepSeek API 调用，支持同步和流式响应
- 来源引用标注（文件名 + 段落位置）
- 多轮对话历史支持
- 超时重试机制（指数退避，3次重试）

### 系统特性
- 启动时从 ChromaDB 自动重建 BM25 索引
- FastAPI CORS 跨域支持
- 环境变量配置（.env 文件）

| POST | /chat | 问答（支持流式） |
| GET | /stats | 知识库统计 |
